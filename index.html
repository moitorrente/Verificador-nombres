Información

Antes de utiliar el programa se deben cargar las listas sobre las que se va a buscar.

Se debe cargar cada una de las listas de manera independiente.

Una vez cargadas se guardarán en el almacenamiento local del navegador. Para eliminarlas se puede realizar utilizando el icono de la basura.

Cuando estén cargadas se debe pulsar sobre → para continugar.
